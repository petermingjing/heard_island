# Output Directory

This directory contains all processed outputs from the Heard Island Glacier Albedo analysis.

## Directory Structure

```
output/
├── *.csv                    # Statistical results
├── VIIRS_slope.tif          # Trend slope raster
├── VIIRS_pvalue.tif         # P-value raster
├── VIIRS_tau.tif            # Mann-Kendall tau raster
└── VIIRS_nobs.tif           # Number of observations raster
```

## Output Files by Analysis

### Descriptive Statistics

- `descriptive_statistics_overall.csv` - Overall statistics
- `descriptive_statistics_seasonal.csv` - Seasonal statistics
- `descriptive_statistics_annual.csv` - Annual statistics
- `descriptive_statistics_monthly.csv` - Monthly statistics
- `summary_statistics_data.csv` - Summary table

### Temporal Trend Analysis

- `time_series_data.csv` - Time series data for visualization
- `trend_analysis_summary.csv` - Summary of trend analysis
- `seasonal_patterns_monthly_data.csv` - Monthly seasonal patterns
- `seasonal_patterns_seasonal_data.csv` - Seasonal patterns

### Spatial Trend Analysis

- `VIIRS_slope.tif` - Pixel-level slope raster (~15 KB)
- `VIIRS_pvalue.tif` - Pixel-level p-value raster (~15 KB)
- `VIIRS_tau.tif` - Mann-Kendall tau raster (~7 KB)
- `VIIRS_nobs.tif` - Number of observations per pixel (~3 KB)
- `VIIRS_trend_map_data.csv` - Trend map data
- `VIIRS_significance_map_data.csv` - Significance map data
- `VIIRS_slope_distribution_data.csv` - Slope distribution
- `VIIRS_elevation_slope_scatter_data.csv` - Elevation-trend relationships
- `VIIRS_elevation_bin_stats.csv` - Elevation quartile statistics
- `VIIRS_elevation_slope_density_data.csv` - Density estimates

### Climate Attribution

- `climate_attribution_monthly_data.csv` - Monthly aggregated data
- `climate_attribution_variable_importance_data.csv` - Variable importance rankings
- `climate_attribution_coefficients_data.csv` - Model coefficients
- `climate_attribution_method_heatmap_data.csv` - Method comparison data
- `climate_correlation_data.csv` - Climate correlation data
- `climate_time_series_data.csv` - Climate time series

### Additional Outputs

- `data_availability_data.csv` - Data availability statistics
- `fig_4_summary_statistics.csv` - Figure 4 summary statistics

## File Formats

### CSV Files

All CSV files use:
- Comma-separated values
- Header row with column names
- No row names
- UTF-8 encoding

### Raster Files (TIF)

Raster files are GeoTIFF format:
- **Projection:** EPSG:4326 (WGS84)
- **Data Type:** Float32
- **No Data Value:** -9999 or NaN

## Regenerating Outputs

All outputs can be regenerated by running the corresponding analysis scripts:

1. **Descriptive Statistics:**
   ```r
   source("scripts/descriptive_statistics_boxplots.R")
   ```

2. **Temporal Trend Analysis:**
   ```r
   source("scripts/albedo_analysis_linear.R")
   ```

3. **Spatial Trend Mapping:**
   ```r
   source("scripts/figure_4_trend_map.R")
   ```

4. **Climate Attribution:**
   ```r
   source("scripts/climate_attribution.R")
   ```

## Key Results Summary

### Temporal Trends
- Linear trend: -0.015 per decade
- Mann-Kendall τ = -0.308, p = 0.161
- 77.3% of glacier pixels show negative trends

### Climate Attribution
- Mean surface solar radiation explains 53.5-60.3% of variance
- Total precipitation is secondary driver
- Significant non-linear effects identified
